what
savvit backs up the pages you have saved on reddit. it exports them to a bookmarks file that you can import in to your web browser or view as a regular html file.

how
savvit is a tiny python script that uses praw, flask, twisted, and a few other packages. you can find the source here.

limitations
chrome has a bug that prevents it from loading the bookmarks file, but a bug report has been filed and it's being fixed. only firefox has been tested to work for importing bookmarks, but you can use any browser to view the file as regular html.
